使用backbone.js和django做的一个小练习，体验下backbone开发项目的感觉，虽然只是练习。


## 项目依赖:

- python2.7
- django1.3.1
- backbone.js


## 使用 virtualenv 运行项目:

```bash

virtualenv --python=python2.7 --no-site-packages ./py2env

source ./py2env/bin/activate

pip2 install django==1.3.1


```
